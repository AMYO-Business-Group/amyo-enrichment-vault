# Start Here — AMYO Enrichment Vault

Read this file first in every new AI or human planning session.

## Why this repo exists

This vault is an off-repo ideation and synthesis space for AMYO. It lets ChatGPT, Gemini, Codex, and humans iterate on ideas without polluting the canonical implementation repo.

Use it for:

- TruckMatch MVP Phase 1 planning
- TruckMatch Phase 2 planning
- TruckMatch full platform-grade concepts
- Ethiopia tax and finance ERP research
- opportunity discovery
- process maps
- data requirements
- ADR drafts
- implementation-plan drafts
- promotion packages

Do **not** use it for:

- production code
- canonical migrations
- final ADRs
- secrets
- service-role keys
- personal credentials
- unsourced legal conclusions
